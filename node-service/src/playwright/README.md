# PlaywrightDemo

学习使用Playeright，

实操：

打开B站，模拟搜索，并把前两个内容的视频播放。
给定关键字数组进行遍历搜索

拓展

关键字数组(问题、兴趣爱好、up主姓名)->AI进行拓展拆分->模拟搜索->搜索结果->AI过滤筛选->打开网址模拟视频播放/点赞。


# 安装 Playwright 核心依赖（包含浏览器驱动）
npm install playwright
载并安装 Playwright 所需的 Chromium、Firefox、WebKit 等浏览器二进制文件。
npx playwright install
npm install playwright@latest
